##Problem
-To predict whether a customer subscribes to a term deposit or not based on their banking behaviour.

##Approach
-Used Logistic Regression for binary classification
-Converted categorical data into neumerical form
-Applied preprocessing techniques to prepare data for modeling

##Methodology
-Converted yes/no into numbers 1/0
-Applied One-Hot Encoding
-Split the dataset into training and testing set
-Applied StandardScaler for feature normalizatiion
-Used Logistic Regression model(suitable for binary classification problem)

##Findings
-Accuracy is 0.89
